# Board
<br>
Springboot Gradle, H2 database, Thymeleaf 사용<br>

### 완성<br>
- 회원가입/ 로그인/ 로그아웃
- 게시판 글 등록/ 수정/ 삭제
- 댓글 등록/ 수정/ 삭제
- 등록/수정/LocalDateTime 설정
- 마이페이지 (회원정보수정/ 내 글/ 내 댓글 모아보기)
<br><br>
느낀 점 : 책에 안 나온 부분(마이페이지)까지 완성해서 뿌듯하다!
<br><br>
추가하고싶은 부분 : 관리자페이지(게시글 삭제, 회원 삭제)
  <br>
  <br>
아쉬운 점 : 그대로 폼을 갖다쓴 것이 아쉽다<br>
